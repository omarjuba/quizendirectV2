-------------------INSTALLATION CYPRESS LINUX--------------------------------


1 apt-get install libgtk2.0-0 libgtk-3-0 libgbm-dev libnotify-dev libgconf-2-4 libnss3 libxss1 libasound2 libxtst6 xauth xvfb

2 sudo apt install npm #si vous n'avez pas npm

3 cd racine_du_projet

4 npm init # initialiser le projet, si vous voulez garder les valeur par défaut, appuyez sur <<Entrée>> jusqu'au bout 

5 npm install cypress --save-dev

6 mettre le repertoire QED-TEST dans ./cypress/integration

7 lancer cypress: ./node_modules/.bin/cypress open
